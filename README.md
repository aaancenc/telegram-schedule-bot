# Telegram Schedule Bot

Telegram-бот на Go для просмотра расписания занятий по дням недели. Бот получает команду от пользователя, берёт расписание из PostgreSQL и отправляет его в Telegram.

## Возможности

- Просмотр расписания по командам `/понедельник`, `/вторник`, `/среда`, `/четверг`, `/пятница`, `/суббота`, `/воскресенье`
- Кнопки с днями недели
- Хранение расписания в PostgreSQL
- Получение предмета, времени, преподавателя и кабинета

## Технологии

- Go
- PostgreSQL
- pgx
- Telegram Bot API

## Установка

Склонируйте репозиторий:

```bash
git clone https://github.com/your-username/telegram-schedule-bot.git
cd telegram-schedule-bot
```

Установите зависимости:

```bash
go mod tidy
```

## Настройка PostgreSQL

Создайте таблицу расписания:

```sql
CREATE TABLE schedule (
    id SERIAL PRIMARY KEY,
    day TEXT NOT NULL,
    time TEXT NOT NULL,
    subject TEXT NOT NULL,
    teacher TEXT,
    room TEXT
);
```

## Настройка Telegram-бота

Создайте бота через `@BotFather` в Telegram:

1. Напишите `/newbot`
2. Укажите название бота
3. Укажите username, который заканчивается на `bot`
4. Скопируйте токен

В коде укажите токен бота:

```go
bot, err := tgbotapi.NewBotAPI("YOUR_TELEGRAM_BOT_TOKEN")
```

Также укажите строку подключения к PostgreSQL:

```go
conn, err := pgx.Connect(ctx, "postgres://username:password@localhost:5432/postgres")
```

Пример:

```go
conn, err := pgx.Connect(ctx, "postgres://andruhablatnoy:12345@localhost:5432/postgres")
```

## Запуск

Запустите проект:

```bash
go run main.go
```

После запуска откройте бота в Telegram и отправьте:

```text
/start
```

Затем выберите день недели кнопкой или отправьте команду:

```text
/понедельник
```

## Безопасность

Не храните токен Telegram-бота и пароль от PostgreSQL в публичном репозитории. Лучше вынести их в `.env` файл и добавить `.env` в `.gitignore`.

Пример `.gitignore`:

```gitignore
.env
```
